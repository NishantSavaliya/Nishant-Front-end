Q-02 : What are the disadvantages of CSS?

Ans : The disadvantages of css are...

-> CSS not support in all browser, so all time need to write special css code.

-> CSS is complex if you are starting, must be idea about selectors, properties, values etc..

-> Developers need to be careful when using CSS to avoid security risks.

-> Complex layouts and limited layout control.

-> CSS files can make web pages load. To keep website performing best, it must be keep CSS as minimum.

-> CSS is “cascading style sheet” meaning styles can pile up and sometimes override each other in unexpected ways.

-> CSS become complex and time-consuming. It is need to best practices and use naming conventions to keep styles manageable.
