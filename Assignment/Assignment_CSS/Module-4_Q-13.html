Q-13 : What are the ways to integrate CSS as a web page?

Ans :

-> There are three ways of inserting a style sheet:
        1. External CSS
        2. Internal CSS
        3. Inline CSS

1. External css

-> With an external style sheet, you can change entire website by changing one file.

-> Each HTML page include a reference in the head section to the external style sheet file inside the <link> element.

2. Internal css

-> An internal style sheet use in single HTML page.

-> The internal style is defined inside the <style> element, inside the head section.

3. Inline css

-> An inline style used to apply a unique style for a single element.

-> To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.
