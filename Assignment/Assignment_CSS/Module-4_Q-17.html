Q-17 : What is the meaning of the CSS selector?

Ans :

A CSS selector selects the HTML element(s) you want to style.

1. ID selector

-> The id of an element is unique within a page, so the id selector is used to select one unique element!

-> To select an element using hash (#) character

For example :- 

        #header {
            text-align: center;
            color: red;
          }

2. class selector

-> The class selector selects HTML elements with a specific class attribute.

-> To select an element using dot (.) character

For example :- 

        .header {
            text-align: center;
            color: red;
          }

3. Universal Selector

-> The universal selector (*) selects all HTML elements on the page.

For example :- 

        * {
            text-align: center;
            color: blue;
          }

4. Group selector

-> The group selector selects all the elements with the same time.

For Example :- 

        h1, h2, p {
            text-align: center;
            color: red;
          }
