Q-1: Are the HTML tags and elements the same things?

Ans: No, They are not sane.

HTML tags are the keywords on a web page that define how your web browser must format and display your web page.
For Ex. <body> is the opening tag and </body> is the closing tag.

HTMl elements includes start tag, content and end tags.
For Ex. '<p> This is apple </p>' In this,<p>-Start tag, This is apple-Content, </p>-End Tag.
