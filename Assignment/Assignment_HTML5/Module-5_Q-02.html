Q-02 : How to embed audio and video in a webpage?

Ans :

-> Add <audio> and <video> elements to the page; make them display the default browser controls.

-> Give both of them <source> elements so that browsers will find the audio format they support best and load it

-> audio is embed in webpage by using <audio> tag.

-> For example -

            <audio>
                  <source src="file name" type="">
            </audio>

-> vidio is embed in webpage by using <vidio> tag.

p>-> For example -

            <vidio>
                  <source src="file name" type="">
            </vidio>
