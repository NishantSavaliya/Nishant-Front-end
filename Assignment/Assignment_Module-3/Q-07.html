Q-7: What is difference between 'id' attribute and the 'class' attribute in HTML elements? With example.

Ans: A Class name can be used by multiple HTML elements, while an ID name must only be used by one HTML element within the page.

For ex.-

<!-- An element with a unique id -->
< h1 id="myHeader" > Tops career Center </h1>

<!-- Multiple elements with same class -->

< h2 class="branch" > Rajkot </h2>
< p > Tops career Center - Rajkot </p>

< h2 class="branch" > Ahmedabad </h2>
< p > Tops career Center - Ahmedabad </p>

< h2 class="city" > Surat </h2>
< p > Tops career Center - Surat </p>
