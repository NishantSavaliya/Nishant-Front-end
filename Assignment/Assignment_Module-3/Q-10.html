Q-10: How can we club two or more rows or columns into a single row or column in an HTML table? With Example.



Ans: A two or more rows club using 'rowspan= (Enter no. of rows)' Attribute.
         A two or more columns club using 'colspan= (Enter no. of columns)' Attribute.


For ex.-
rowspan:
A B
C D
E
F G

colspan:
A B
C
E F
G H
