To apply css

            $('button').click(()=>{
                  $('div').css({'background-color': 'yellow', 'font-size':'1.5em'});
            })

To add class and remove class

child-2
            *********** Add class ****************

            $(".add").click(function(){
                  $(".parent").addClass("bg-color");
            });
            
            *********** Remove class ****************

            $(".remove").click(function(){
                  $(".parent").removeClass("bg-color");
            });

JQuery Animation

-> The jQuery animate() method is used to create custom animations.

            $("button").click(function(){
                  $("div").animate({
                        opacity: '0.5',
                        height: '300px',
                        width: '500px'
                  });
            });
