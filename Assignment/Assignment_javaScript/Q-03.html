Q-03 : Define a Data Types in js?

Ans : 

JavaScript has 8 Datatypes
            1. String
            2. Number
            3. Bigint
            4. Boolean
            5. Undefined
            6. Null
            7. Symbol
            8. Object

For Ex-

// Numbers:
            let length = 16;
            let weight = 7.5;
            
            // Strings:
            let color = "Yellow";
            let lastName = "Johnson";
            
            // Booleans
            let x = true;
            let y = false;
            
            // Object:
            const person = {firstName:"John", lastName:"Doe"};
