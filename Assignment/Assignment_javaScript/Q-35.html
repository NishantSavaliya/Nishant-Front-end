Q-35 : Write a program make a summation of given number (E.g., 1523 Ans: - 11) in JS?

Ans : 

            function sumOfDigit(num) { 
                  let numStr = num.toString(); 
                  let sum = 0; 
                  
                  for (let digit of numStr) { 
                        sum += parseInt(digit); 
                  } 
                  
                  return sum; 
            } 
      
            console.log(sumOfDigit(123));
