Group Selector


-> Group selector select using ','.

-> Group selector is use to select multiple element of that webpage.

For Example:
<p> </p>
<div> </div>

- Apply style using internal css.

            <style>
                h2{
                    color: blue;
                    font-size: 20px;
                }
        
                p, div{
                    color: brown;
                    background-color: antiquewhite;
                    padding: 10px;
                }
            </style>
